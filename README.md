# Hire GenAI

A Next.js-powered hiring assistant platform built with v0. Includes candidate management, job postings, messaging, talent pool, delegation, and analytics dashboards.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000. The root redirects to /dashboard.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Lucide icons
- Vercel Analytics

## Project Structure

- `app/` – Next.js app router pages and layouts
  - `dashboard/` – Main app dashboard with sub-routes
- `components/` – Reusable UI components
- `lib/` – Utilities and helpers
- `hooks/` – Custom React hooks
- `public/` – Static assets
